Go
===================

Go est un langage de programmation compilé et concurrent inspiré de C et Pascal.

Utilité :
::

  Serveurs
  Web
  Systèmes embarqués
  IOT (Internet Of Things)
  Android
  IOS
  Jeux-vidéos

Pré-requis :
===================

  Visual Studio Code avec l'extension Go
  Le Complilateur Go :   `Install Complilateur Go`_

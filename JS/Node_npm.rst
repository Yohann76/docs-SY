
Node.JS & npm
===================

Checker les dépendance npm non utilisé :
-------------------

::

  npm install -g npm-check
  
::

  npm-check
  
Mettre a jour npm
--------------------

::

  npm install -g npm
  
Démarrer un projet npm ( avec les set config pré-rempli )
-----------------------

::

  npm init
  
Chercher un package avec son nom partiel 
----------------------------

::

  npm search <nompartiel>
